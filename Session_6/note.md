- naming convention:
 + no number in the beginning
 + CamelCase, Snake_Case, PascalCase
 + Primitive data value:
  * String ("Hello world")
  * Number (100)
  * Boolean (true/false)
  * Null
  * Undefined
 + Composite/Non-premitive data value:
  * Object
  * Array
